# Make-Planner

Esse site será desenvolvido com o interesse de por conhecimentos em prática e para treino.
A ideia principal é que os usuarios possam criar e editar seus proprios planners, mas não apenas isso, criar listas de tarefas ou rotina de estudos. O público alvo seriam aqueles estudadantes que gostam de enfeitar suas notas e afazeres, ou precisam de apenas um ambiente para se organizar.
